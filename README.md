# 🎉 隨機抽獎網頁

一個簡潔易用的隨機抽獎工具，使用純 HTML、CSS 和 JavaScript 開發。

## 功能特性

- ✨ **自訂抽獎範圍** - 輸入最小值和最大值（預設 1~100）
- 🎲 **隨機抽獎** - 點擊按鈕從指定範圍內隨機抽取一個數字
- 🎬 **抽獎動畫** - 在結果區域顯示 2 秒的抽獎中動畫效果
- 📊 **結果記錄** - 自動記錄已抽出的數字，一列 5 個排列
- 🔒 **防止重複** - 不會重複抽到相同的數字
- 🧹 **清除記錄** - 可一鍵清除所有抽獎記錄，重新開始

## 使用方法

1. 用瀏覽器直接開啟 `index.html` 檔案
2. 在上方的輸入框中設定要抽獎的數字範圍
3. 點擊「開始抽獎」按鈕
4. 等待 2 秒的動畫效果，結果會以紅色大字顯示
5. 已抽出的數字會自動記錄在下方
6. 點擊「清除記錄」按鈕可重新開始

## 技術棧

- **HTML** - 頁面結構
- **CSS** - 樣式設計與動畫效果
- **JavaScript** - 抽獎邏輯

無需任何外部套件或依賴，可獨立運作。
